# Network Graph Visualization Demo

A simple interactive network diagram built with React + D3, 
showcasing server-router connections with 
click-to-open node menus.

## Features
- Interactive network graph with nodes (1 Server + n Routers)
- Click any node to open a menu (Inspect, Info)
- Custom styled edges with arrows and labels
- Drag, zoom, and pan controls

## Tech Stack
- React + Vite


##  Installation & Setup

### Pre-Requirements
- Node.js (v20+ recommended)
- npm or yarn
- React 18 (no React 19 support yet)

### 1. Clone & Install
## > git clone ....
## > cd Eyeball-View-DemoD3
## > npm install


### 2. Run Development Server 
## > npm run dev
